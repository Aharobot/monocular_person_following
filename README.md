# monocular_person_following

This package provides a monocular vision-based person tracking and identification framework for person following robots. 

## Dependencies

- tf-pose-estimation
- monocular_people_tracking
- ccf_person_identification


## Related packages

- [ccf_person_identification](https://github.com/koide3/ccf_person_identification)
